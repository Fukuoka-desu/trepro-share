第38章

学び残しの地図を、自分の手で描く Coverage Map

『本書のどこを、自分はまだ読めていないのか』。 これは記憶の問題ではなく、地図の問題です。

ほとんどの人は、教科書を読み終えた後で『学び残し』を感覚で扱っています。 ですが、章ごとの Coverage を地図として持てば、未到達の箇所は一目で分かります。

この章で扱うのは、本書 40 章＋14 レッスンの Coverage Map です。 seen / practiced / verified / applied の 4 段階で、自分の現在地を可視化します。

学び残しが、地図で一目でわかる。次に何へ手を伸ばすべきかが、迷わず決まります。
